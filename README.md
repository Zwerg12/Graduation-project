## Выпускной проект

### Описание проекта

- Провести исследовательский анализ данных интернет-магазина товаров для дома;
- Проанализировать торговый ассортимент;
- Сформулировать и проверить статистические гипотезы.

### Цель проекта:
предложить алгоритм разделения всех товаров на основной и дополнительный ассортимент

### Описание данных:

- `date` — дата заказа;
- `customer_id` — идентификатор покупателя;
- `order_id` — идентификатор заказа;
- `product` — наименование товара;
- `quantity` — количество товара в заказе;
- `price` — цена товара.

### Использованные в проекте библиотеки и методы:
pandas,
numpy,
scipy,
pymystem3, 
Counter
